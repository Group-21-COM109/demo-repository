# Bloom & Co. - Flower Shop Website

A contemporary and accessible flower shop website built for COM109 Client Side Development coursework at Ulster University.

## Project Overview

This project is a group assignment creating an interactive flower shop website with:
- Modern, accessible design
- Web form with extensive validation
- JavaScript/jQuery animations
- Responsive layout

## Features

- **Home Page**: Showcase of flowers with animated reveals
- **Contact Form**: Comprehensive form validation (name, email, phone, address, postcode)
- **Accessibility**: WCAG compliant design with skip links, ARIA labels, keyboard navigation
- **Responsive**: Works on all devices

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery

## Folder Structure

```
demo-repository/
├── index.html           # Home page
├── contact.html         # Contact form page
├── css/
│   └── style.css      # External stylesheet
├── js/
│   ├── main.js        # Animation and interactivity
│   └── validation.js  # Form validation
├── images/
│   ├── flowers/       # Flower images (add your own)
│   ├── background.jpg # Background image (add your own)
│   └── favicon.ico   # Website icon (add your own)

```

## Getting Started

1. Clone the repository
2. Add your images to the `images/` folder:
   - `background.jpg` - Background image for hero section
   - `favicon.ico` - Website icon
   - `flowers/roses.jpg` - Rose image
   - `flowers/sunflowers.jpg` - Sunflower image
   - `flowers/lilies.jpg` - Lily image
   - `flowers/tulips.jpg` - Tulip image
3. Open `index.html` in a web browser
